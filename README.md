# didii Payment Flow Demo

A polished 3-screen fintech frontend prototype inspired by the Product Engineer, Frontend role at didii.

This project demonstrates a mobile-first payment experience built with React Native, Expo, and TypeScript. It focuses on conversational UI, payment confirmation, and photo-to-payment interaction patterns for Nigerian users.

## Screens

### 1. Conversational Assistant UI
A chat-style money assistant where the user can initiate a payment naturally.

### 2. Payment Confirmation Flow
A clean confirmation screen showing recipient details, amount, transfer fee, delivery speed, and payment success feedback.

### 3. Photo-to-Payment Upload Mock
A mock OCR-style interface where users can upload a receipt, bill, or screenshot and automatically extract payment details.

## Tech Stack

- React Native
